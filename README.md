## Description
This program calculates each student’s attendance percentage based on 30 working days and determines their exam eligibility (75% or above).
It also counts how many students are eligible or not and computes the average attendance of all students.
Finally, it displays a summary showing total students, eligible count, not eligible count, and the average attendance percentage.
